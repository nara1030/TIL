## 요약
### 디자인 패턴 개요
보통 책에서 언급하는 디자인 패턴은 아래와 같고, 이외에는 곁가지라 해도 무방하다.

1. [템플릿 메소드 패턴](object1_ch10.md): 상속(`좋은 상속`)
2. 전략 패턴: 구성(조립)
3. 컴포지트 패턴: 파일 탐색기
	* 비지터 패턴: 제어 위임
4. 데코레이터 패턴
5. Chain of Responsebility
6. 커맨드 패턴
